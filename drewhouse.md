1. Project Overview

    * Goal: Create a high-end, minimalist portfolio website for a top-tier video editor
    * Primary Action: Highlighting project work (video) and converting visitors into inquiries
    * Target Audience: Film producers, ad agencies, and high-stakes commercial clients

2. Core Features & Requirements

Hero Section - A full-screen "Showreel" video background (muted) with bold, high-contrast headline
Project Grid - A "Bento Grid" or minimalist masonry layout. Each project card features a hover-to-play video preview
Project Details - Individual pages for major projects including: Client name, role (Editor/Colorist), software ised. and a brief "process" description
Video Integration - Seamless embedding from Youtube, using a custom-styled player
About Section - A clean layout with a professional headshot, a list of "Big Names" worked with, and a link to a downloadable PDF Resume
Contact Form - A simple, elegant form (Name, Email, Project Type, Message)

3. Design Principles

    * Minimalism: Heavy use of whitespace to let the colors in the videos pop
    * Dark Mode by Default: Video work looks significantly better on deep charcoal or black backgrounds
    * Micro-interactions: Subtle transitions when scrolling and smooth "fade-in" effect for text
    * Typography: Large, bold sans-serif headers (e.g, Inter or PP Neue Montreal) to convey authority

4. Tech Stack

Framework: Next.js (react)
Styling: Tailwind CSS
Animations: Framer Motion
Video Hosting: Youtube
